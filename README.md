### Installation

NPM:

```bash
    npm install @ing3kth/core
```

Yarn:

```bash
    yarn add @ing3kth/core
```

### Usage

```javascript
const IngCore = require("@ing3kth/core");
```

```typescript
import * as IngCore from "@ing3kth/core";
```
